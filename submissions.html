Submissions

Submissions are currently open for the first edition, only black and white visual work will be accepted. Written pieces should be 4 pages or less in length (font size and layout makes a difference, if it's close to it I'll make it work).

When submitting you will need to provide these informations:

  • Name or pseudodym
  • Link to website or social media
  • Page format desired (small spread, large single page, etc.)
  • Paper color desired
  • Text file if written piece
  • Image file (min 300 dpi) for visual works
  • Accompanying text for visual works (optional, max 200 words)

Please check the details page for more information on format and paper colors.

Send your submission to fantomeszine @ gmail (dot) com
